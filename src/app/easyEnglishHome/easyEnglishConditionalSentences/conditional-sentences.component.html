Conditional Sentences

Conditional sentences are sentences that express one thing contingent on something else, e.g. "If it rains, the picnic will be cancelled". They are so called because the impact of the main clause is conditional on the dependent clause.

Used for general truths and facts. Structure: If + present simple, present simple.
Example: If you heat ice, it melts.

Used for real possibilities in the future. Structure: If + present simple, will + infinitive.
Example: If it rains tomorrow, we will cancel the trip.

Used for unreal or improbable situations in the present or future. Structure: If + past simple, would + infinitive.
Example: If I won the lottery, I would buy a car.
Another example: If I had more time, I would travel the world.

Used for unreal or improbable situations in the past. Structure: If + past perfect, would have + past participle.
Example: If I had known you were in town, I would have called you.
